# 麻花管几何说明图示

## 图片文件位置

请将麻花管几何说明图示文件命名为 `plum-blossom-diagram.png`，并放置在 `public/` 目录下。

文件路径：`public/plum-blossom-diagram.png`

## 图示说明

图示应包含以下内容：
- **外圆（黑色）**：代表直径34mm的外管内径
- **梅花形状（红色）**：内切的6瓣梅花形状
- **空隙区域（天蓝色）**：外圆与梅花形状之间的6个空隙
- **深度标注**：3mm（从外圆表面到梅花凹槽底部）
- **计算值**：
  - 总空隙面积：~149.62 mm²
  - 等效圆直径：~13.80 mm

## 显示位置

图示会在以下位置显示：
- 麻花管参数输入区域
- 当选择"麻花管"类型时可见
- 如果图片文件不存在，会显示文字说明作为替代

## 空隙面积计算逻辑

代码中统一使用以下计算方法：

```javascript
空隙面积 = 外圆面积 - 梅花形状面积
```

### 几何定义

**外圆面积**：
- 定义：外管内壁形成的圆形面积
- 公式：`外圆面积 = π × (外管内径/2)²`
- 说明：外管内径 = 外管名义内径 + 1mm安装间隙

**梅花形状面积**：
- 定义：麻花管外轮廓形成的梅花截面面积（从峰顶到谷底）
- 计算：`梅花形状面积 = calculateLobeCrossSection(doMax, doMin, lobeCount).area`
- 参数说明：
  - `doMax`：麻花管外径（峰顶直径，等于外管名义内径）
  - `doMin`：麻花管谷底直径（峰顶直径 - 2×齿高）
  - `lobeCount`：头数（瓣数，通常3-6）

**环隙流通面积**：
- 定义：外管内壁与麻花管外轮廓之间的空隙面积
- 公式：`环隙流通面积 = 外圆面积 - 梅花形状面积`
- 修正：考虑螺旋通道的实际有效流通面积，应用螺旋通道修正系数

### 应用范围

此计算方法已统一应用于：
1. 传热系数计算中的环形空间面积
2. 压降计算中的环形空间面积
3. 几何参数计算中的环隙当量直径
